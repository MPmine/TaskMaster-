Документация API

Пользователи

  • GET /api/users — получить список всех пользователей
  • POST /api/users — создать нового пользователя
    • Параметры: username, email, password

Задачи

  • GET /api/tasks — получить все задачи
  • GET /api/users//tasks — задачи конкретного пользователя
  • POST /api/tasks — создать новую задачу
    • Параметры: title, description, status, priority, due_date (ISO формат), user_id
  • GET /api/tasks/ — получить задачу по ID
  • PUT /api/tasks/ — обновить задачу (передаются только изменяемые поля)
  • DELETE /api/tasks/ — удалить задачу

Примеры запросов

# Получить всех пользователей
curl -X GET http://localhost:5000/api/users

# Создать задачу
curl -X POST http://localhost:5000/api/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Новая задача",
    "description": "Описание задачи",
    "status": "pending",
    "priority": 3,
    "due_date": "2024-12-31T23:59:59",
    "user_id": 1
  }'

Коды ответов

  • 200 — Успешно
  • 201 — Создано
  • 400 — Неверный запрос
  • 404 — Ресурс не найден
  • 500 — Внутренняя ошибка сервера
